[图片]

    使用img单标签可以在网页中插入图片，还可以设置显示图片的长宽比

属性一览
  • src:图片的地址，尽量用相对路径，此标签必须有！
  • alt：替换文本，当图片显示不出来的时候显示的文字，此标签必须有！
  • title：提示文本，鼠标悬停在图片上面时显示的文字
  • width and heigth：图片的宽度和高度(数字)
    1. 如果之设置其中一个值，系统会自动调整比例
    2. 如果两个值都自行设置，要注意图片会不会变形
  • border: 设置边框，数值为数字

[超链接（重要）]

超链接可以是一个字，一个词，或者一组词，也可以是一幅图像，您可以点击这些内容来跳转到新的文档或者当前文档中的某个部分。
当您把鼠标指针移动到网页中的某个链接上时，箭头会变为一只小手。
在a标签中使用了href属性来描述链接的地址。

属性
  • href->链接地址，可以是一个锚点也可以是网址或者html文件，必须要有的属性;
    如果要链接一个图片使用img标签即可
  • target->设置链接的打开方式
    1. _self:默认方式，即在当前窗口打开链接
    2. _blank:在一个全新的空白窗口中打开链接
    3. _top:在顶层框架中打开链接
    4. _parent:在当前框架的上一层里打开链接
    5. 我们只需要掌握“_self”和“_blank”这两个属性值就可以了，其他两个用不到。
  • id->设置书签，可以用锚点书签来链接

锚点链接

 《a href = "#书签名称" 》链接文字《/a》

可以跳转到连接的书签，包涵id的div,a标签中
